# 休息提醒 - Rest Reminder

定时提醒你休息，保护眼睛和身体健康的 VSCode 插件。

## ✨ 功能特性

- ⏰ 可自定义提醒间隔（5-120 分钟）
- 🌙 夜间模式界面，保护眼睛
- 🖼️ 支持自定义网络图片
- 📊 详细的休息统计和可视化图表
- 🎨 三种提醒样式：紧凑/完整/通知
- ⚙️ 丰富的配置选项
- 💾 自动保存统计数据

## 🚀 快速开始

1. 安装插件
2. 插件会自动启动提醒
3. 按 `Ctrl+Shift+P` 输入 `休息` 查看所有命令

## ⚙️ 配置

在设置中搜索 `restReminder` 进行配置：

- `restReminder.interval`: 提醒间隔（分钟）
- `restReminder.reminderStyle`: 提醒样式
- `restReminder.autoCloseTime`: 自动关闭时间
- `restReminder.imageUrl`: 自定义图片 URL
- ...更多配置

## 📝 使用建议

- 建议设置 25-30 分钟提醒一次
- 休息时远眺窗外至少 20 秒
- 适当活动身体，避免久坐
